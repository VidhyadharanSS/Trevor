# Keyboard Shortcuts

Every action in Trevor is reachable from the keyboard. The reference below mirrors what you'll see if you press **Mod + /** inside the app.

> **Mod** is **Cmd** on macOS, **Ctrl** on Linux and Windows.

---

## Notes

| Shortcut | Action |
| --- | --- |
| Mod + N | New note in current folder |
| Mod + Shift + N | New note at vault root |
| Mod + S | Save active note |
| Mod + W | Close active note |
| Mod + [ | Back — previous note in history |
| Mod + ] | Forward — next note in history |
| F2 | Rename selected note |
| Del | Move selected note(s) to trash |
| Mod + D | Duplicate active note |

## Editor — formatting

| Shortcut | Action |
| --- | --- |
| Mod + B | Bold |
| Mod + I | Italic |
| Mod + U | Underline (HTML `<u>`) |
| Mod + E | Inline code |
| Mod + Shift + X | Strikethrough |
| Mod + K | Insert link |
| Mod + Alt + 1..6 | Heading 1..6 |
| Mod + Alt + 0 | Plain paragraph |
| Mod + Shift + L | Bulleted list |
| Mod + Shift + O | Numbered list |
| Mod + Shift + T | Task list |
| Mod + Shift + Q | Block-quote |
| Mod + Shift + C | Fenced code block |
| Mod + Shift + M | Math block |

## Editor — navigation

| Shortcut | Action |
| --- | --- |
| Mod + L | Go to line |
| Mod + Home | Top of note |
| Mod + End | Bottom of note |
| Tab | Indent / expand snippet |
| Shift + Tab | Outdent |
| Mod + / | Toggle line comment |

## Search

| Shortcut | Action |
| --- | --- |
| Mod + K | Open command palette (titles, tags, commands, content) |
| Mod + O | Quick switcher (titles only) |
| Mod + F | Find in current note |
| Mod + G | Find next |
| Mod + Shift + G | Find previous |
| Mod + Shift + F | Vault-wide content search |

## View

| Shortcut | Action |
| --- | --- |
| Mod + B | Toggle sidebar |
| Mod + Shift + B | Toggle right panel (outline / backlinks) |
| Mod + . | Toggle preview mode |
| Mod + Shift + . | Toggle split view |
| Mod + Shift + G | Toggle graph view |
| Mod + + | Zoom in |
| Mod + - | Zoom out |
| Mod + 0 | Reset zoom |

## Selection & bulk operations

| Shortcut | Action |
| --- | --- |
| Mod + click | Toggle a note in the multi-selection |
| Shift + click | Range-select |
| Esc | Clear selection |
| Drag | Move all selected notes to a folder |
| Del | Trash all selected notes |

## Export & import

| Shortcut | Action |
| --- | --- |
| Mod + Shift + E | Export menu (PDF / HTML / MD / bundle / import) |
| Mod + P | Export current note as PDF |

## Application

| Shortcut | Action |
| --- | --- |
| Mod + , | Open settings |
| Mod + / | Open shortcuts reference |
| Mod + Q | Quit (desktop only) |
| Mod + Shift + R | Reload window |

## Customising shortcuts

Trevor's keymap is fixed for now to keep muscle memory consistent across installations. Custom keymaps are tracked in [issue #N/A](https://github.com/VidhyadharanSS/Trevor/issues) — vote with a thumbs-up if you want them.
